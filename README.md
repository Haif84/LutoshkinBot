# Telegram-бот для тестирования коллектива по методике А.Н. Лутошкина

Проект реализован на Python с использованием библиотеки `python-telegram-bot` версии 20.x и SQLite.

## Структура проекта

- `main.py` – запуск бота, инициализация БД, регистрация хендлеров и запуск polling.
- `config.py` – загрузка и валидация конфигурации (токен бота и пр.).
- `db.py` – работа с локальной базой SQLite (таблицы `admins`, `tests`, `results`).
- `handlers.py` – все хендлеры для `ConversationHandler` и логика состояний.
- `keyboards.py` – генерация reply- и inline-клавиатур.
- `questions.py` – список вопросов тестирования.
- `requirements.txt` – зависимости проекта.

## Установка и запуск

1. **Создайте и активируйте виртуальное окружение (рекомендуется)**:

```bash
python -m venv venv
venv\Scripts\activate  # Windows
```

2. **Установите зависимости**:

```bash
pip install -r requirements.txt
```

3. **Создайте файл `.env` в корне проекта и укажите токен бота**:

```text
BOT_TOKEN=1234567890:ABC-DEF...
```

4. **Добавьте администраторов в таблицу `admins`** (любыми удобными средствами: `sqlite3`, SQLite Browser и т.п.):

```sql
INSERT INTO admins (user_id, username) VALUES (123456789, 'admin_username');
```

База данных `lutoshkin.db` создаётся автоматически при первом запуске (функция `create_tables()`).

5. **Запустите бота**:

```bash
python main.py
```

## Основной функционал

- Стартовый экран предлагает выбрать роль: **Администратор** или **Тестируемый** (ReplyKeyboard).
- **Администратор**:
  - Проверяется наличие `user_id` в таблице `admins`.
  - Меню администратора через InlineKeyboard:
    - Просмотр текущих активных тестирований (код, описание, дата).
    - Создание нового тестирования (запрос описания, генерация уникального 6-значного кода, сохранение в БД).
- **Тестируемый**:
  - Ввод 6-значного кода тестирования и проверка его активности.
  - Ввод фамилии.
  - Последовательное прохождение 5 вопросов с ответами 1–5 (InlineKeyboard).
  - Сохранение ответов в таблицу `results` в виде JSON.
- В диалогах предусмотрены кнопки **«Назад»** и **«Отмена»** для возврата и отмены.
